AP ローカルスイッチング　FlexConnectでのみ利用できる機能。APがVLAN番号を付加することでWLCなしでデータ転送を行う。
Sniffer パケットキャプチャの取得ができる。
Bridge　距離が離れたAPの中継ができる。

show ip ospf neighborの最後に表示されるinterfaceは自身の接続しているポート

eBGP(External)　ルーティングプロトコルの１つ　AD値は20　BGPパケットのTTLが１のためBGPネイバーを確立するためルータ間で直接接続している必要がある。
iBGP(Internal)　ルーティングプロトコルの１つ　AD値は200　BGPパケットのTTLが255のため直接接続でなくてもBGPネイバーを確立できる。

信頼境界(Trust Boundary)　QoS対象となるパケットがネットワークに入ってくるエッジのこと。アクセススイッチやIP Phoneなどの境界がコレ

NTP認証　不正な時刻同期を防ぐための認証　
(config)#ntp authenticate　　NTP認証の有効化
(config)#ntp authentivation-key [鍵番号] md5 [パスワード]　　ntp認証で使用できる鍵を作成
(config)#ntp trusted-key {鍵番号}　作成したカギをNTP認証に使用できるように有効化
上記コマンドをクライアントとサーバ両方で入力する。クライアントだけは事前に
(config)#ntp server [サーバのアドレス] key [鍵番号]を入力しておく

VLANにて　ネイティブVLANが異なる場合でもトランクリンクは形成可能。ただし、VLANIDの異なるタグなしのパケット同士でのデータやり取りとなる。

JSON(JavaScript Object Notation)　テキストデータの一種。人間にもプログラムにも扱いやすいため、NBIでRESTful APIとプログラムの間で利用される。

Total output drops　テールドロップが発生したときに増加するカウンタ。show interfaceで確認できる。
ignored　はバッファが足りずに無視したフレームの数で、ブロードキャスト時に増加する。

DAI(Dynamic ARP Inspection)　untrustedのポート届いたARPを検査し、DHCPスヌーピングバインディングデータベースや手動で作成したデータベースを確認してIPアドレスとMACアドレスの対応が正しくなければパケットを破棄する。PCなどを接続するポートは攻撃者が接続している可能性があるのでuntrudtedにするべき。

(config)#logging monitor [レベル]　　vty接続しているPCにsyslogを出力させる
(config)#logginh buffered　[バッファサイズ] [レベル]　　デバイス自身に出力させる
(config)#logging console [レベル]　　コンソールケーブル接続しているPCに出力させる
(config)#logging trap [レベル]　外部のサーバや保存装置に出力

(config)#ip default-gateway [ネクストホップ]　スイッチのVLANにIPアドレスを設定したとき、デフォルトゲートウェイを設定するためのコマンド
(config)#ip route 0.0.0.0 0.0.0.0 [ネクストホップ]　コレはスタティックルーティングでデフォルトルートを設定するコマンドだが、スイッチには使えない。

Ansibleの管理対象となるデバイスはマネージドノード。　管理対象となる機器を定義したファイルはイベントリ。処理内容の記載はプレイブック。

Cisco DNA Center　AIによる情報収集によって障害の予兆を検知することは可能だが、AIによる自動復旧はできない。

スティッキーラーニング　学習したセキュアMACアドレスはrunning-configに保存される

リフレクション攻撃　送信元を偽った要求をサーバに送り、ターゲットに大量の応答を返すように仕向ける攻撃。(増幅攻撃やアンプ攻撃とも)

ACLはI/Fに適用するときはip access-group、vtyに設定するときはip access-class

完全指定スタティックルートは出力I/Fの後にネクストホップ。　IPv6でも同様。

show ip route ではフローティングスタティックルートが表示されない。「S」がスタティックルートルート　＊がデフォルトルート
show ip static routeならフローティングも含めたすべてのスタティックルートが確認できる　「A」がスタティックルート　デフォルトルートは0.0.0.0が送信元

アシュアランス　Cisco DNA Centerの機能の一つ。ネットワークのモニタリングを行う。SNMPやsyslog,NetFlowなどの様々なソースからデータを収集し、ネットワークを監視,分析することで問題の早期発見や解決が可能となる。

コントロールプレーンの役割
・ARPによるテーブル構築
・ルーティングプロトコルによるMACアドレステーブル追加
・IPv6のNDP(Neighbor Discovery Protocol)

FTP ポート番号20はデータ転送用　21番は制御用

IEEE802.11　(無線環境)では管理フレーム,制御フレーム,データフレームを扱う。

ロードバランシング　負荷分散のこと。スタティックでは、同じ宛先への経路を異なるネクストホップで複数作成することでロードバランシング適用するが、ダイナミックルーティングであれば特に設定しなくても有効になる。特にEIGRPは不等コストロードバランシングといってメトリックが異なっていてもok。
