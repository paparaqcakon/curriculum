URIが上記の時
http　使用するプロトコル
api.example.com　APIを提供するホスト名　もしくはIPアドレス
/api/v1/system/hostname　APIのリソース名
ip=10.0.0.1&region=east　APIに渡すパラメータ

GRE over IPsec VPN　はIPsecの通信モードではなくただ組わせてVPNを構築する方法のこと。
トランスポートモード　エンドツーエンドの通信時ではこちらを利用する
トンネルモード　ネットワーク間でのVPNで利用する。

Fast SSID Change あるSSIDに接続しているクライアントが別のSSIDに素早く切り替える機能。デフォルトでは無効。
Fast Transition　高速ローミングを可能にする機能。

clock set 22:11:00 8 Jun 2021　システムクロックの手動設定
calendar set 22:11:00 30 10 2021　ハードウェアクロックの手動設定

SNMP　
NMS(Network Management Station)
Get Request　指定オブジェクトIDに対応した情報の要求
GetNext Request 　次のオブジェクトIDに対応した情報の要求
GetBulk Request　 繰り返し情報の要求
Set Request　SNMPエージェントの制御
Get　Response　マネージャからの要求に対する応答
Trap　SNMPエージェントからマネージャへの一方的な状態通知
Inform　Request　SNMPエージェントからマネージャへの状態通知。応答がなければ再送する。

HSRP ネゴシエーションの状態
Initial HSRPがまだ動作していない(設定変更またはインタフェース起動直後の状態)
Learn　仮想IP未設定、アクティブルータからのHelloパケット待ち
Listen　仮想IPアドレス決定
Speak　自身のHSRP状態を通知
Standby　次にActiveになる候補
Active　仮想IPアドレス宛てのパケットを処理する
